Ejercicio 4

Tenemos el siguiente array de puntuaciones de usuarios: let puntuaciones = [ { nombre: "Pablo", puntuacion: 180 }, { nombre: "Javier", puntuacion: 270 }, { nombre: "Raquel", puntuacion: 70 }, { nombre: "Mario", puntuacion: 310 }, { nombre: "Miriam", puntuacion: 90 }, { nombre: "Fernando", puntuacion: 105 }, { nombre: "Laura", puntuacion: 210 }, { nombre: "Julian", puntuacion: 520 }, ];

Cada elemento del array es un objeto que contiene el nombre y la puntuación total. Se pide mostrar por consola cada elemento del array dado. Implementar una función jugadoresTop(array) que reciba dicho array y devuelva un nuevo array (sin modificar el que se pasa por parámetro) con los nombres de aquellos jugadores que hayan sacado más de 100 puntos. Finalmente muestra por consola cada elemento del nuevo array. Recomendación: en la implementación de la función cuando declaréis el nuevo array podéis usar el método filter() y pasarle como parámetro una función flecha en la que obtengáis los elementos del array con la puntuación mayor que 100.
