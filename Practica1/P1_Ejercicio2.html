Ejercicio 2

La empresa Entretenimientos del Sur S.L. está trabajando en un juego de zombies y nos han pedido que continuemos con su desarrollo. Por ahora tienen una estructura de datos para representar los zombies, que es la siguiente:

class Zombi { constructor(nombre, puntosVida, potencia) { this.nombre = nombre; this.puntosVida = puntosVida; this.potencia = potencia; } atacar(objetivo) { console.log( `${this.nombre} ataca a ${objetivo} y le causa ${this.potencia} puntos de daño` ); } }

Crea un nuevo subtipo de Zombi llamado Abominacion (Abominacion será una subclase de Zombi), que además de funcionar como cualquier Zombi (el constructor de Abominacion debe llamar al constructor de la clase Zombi), tiene un ataque especial llamado ataqueMultiple(objetivo), que ataca 3 veces consecutivas a un adversario. Crea un objeto Abominacion y ejecuta el ataqueMultiple(objetivo).
